# Fetch MCP Server

1. Official [docs](https://github.com/modelcontextprotocol/servers/tree/main/src/fetch)

2. Setup in Visual Studio Code (Docker):
```json
"mcp": {
    "servers": {
      "fetch": {
        "command": "docker",
        "args": ["run", "-i", "--rm", "mcp/fetch"]
      }
    }
  }
```
